# 一个非常小的 HTTP 消息转发服务器

这是一个非常简单的使用 HTTP 进行通信的服务器端；  

这个 Server 的初衷是仅仅是为了进行消息（json数据）的转发；

然后我想，是否还能用来来作为进行图片，视频甚至语音的传输；  

所以我打算日后持续对这个小项目进行优化更新；
